Surface Detection

Towers are placed on surfaces detected by WebXR. The system checks gold costs before creating a tower with specific stats (type, level, HP). Shield towers draw enemy aggro while combat towers automatically target enemies within range. 

fire: function (target) {
  let bullet = document.createElement('a-entity');
  let pos = this.el.object3D.position;
  let data = TOWER_DATA[this.data.type];

  bullet.setAttribute('position', {
    x: pos.x,
    y: pos.y + data.offsetY + 0.05,
    z: pos.z
  });
  bullet.target = target;
  bullet.damage = this.damage;

}

Tower Mechanics

Each tower possesses distinct combat logic. Shield towers distract enemies with high health pools, while offensive towers use timers to trigger attacks. The system also manages 3D positional audio and special effects like stunning.
